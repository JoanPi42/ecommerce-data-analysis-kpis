# 📊 Análisis de Datos E-commerce

## 🧠 Descripción
Este proyecto consiste en un análisis de datos de e-commerce enfocado en la generación de KPIs clave para entender el rendimiento del negocio, el comportamiento del cliente y oportunidades de crecimiento.

Se trabajó con Python utilizando pandas y matplotlib para limpiar, transformar y analizar los datos.

---

## 🎯 Objetivos
- Analizar ventas y crecimiento del negocio  
- Evaluar el comportamiento de compra de los clientes  
- Identificar productos y categorías más relevantes  
- Detectar oportunidades de mejora mediante datos  

---

## 📊 KPIs Analizados

- **Ventas mensuales:** evolución de ingresos a lo largo del tiempo  
- **Crecimiento mensual:** variación porcentual de ventas entre meses  
- **Ticket promedio:** ingreso promedio por pedido  
- **Productos por pedido:** cantidad promedio de artículos por compra  
- **Top productos:** productos con mayor volumen de ventas  
- **Ventas por canal:** desempeño por plataforma de venta  
- **Ventas por ciudad:** distribución geográfica del ingreso  
- **Participación por ciudad:** porcentaje de contribución al total  
- **Tasa de conversión:** proporción de visitas que terminan en compra  

---

## 🛍️ Análisis Avanzado

### Market Basket Analysis
Se aplicó el algoritmo Apriori para identificar relaciones entre productos y detectar patrones de compra.

Esto permite:
- Recomendaciones de productos  
- Estrategias de cross-selling  
- Creación de combos  

---

## 📈 Principales Insights

- Las ventas presentan variaciones a lo largo del tiempo, sin un crecimiento constante.  
- La tasa de conversión es baja (~0.94%), lo que indica oportunidad de mejora.  
- Algunas ciudades concentran una gran parte de las ventas.  
- No existe una categoría claramente dominante, el portafolio está equilibrado.  
- Existen patrones de compra entre productos que pueden aprovecharse comercialmente.
